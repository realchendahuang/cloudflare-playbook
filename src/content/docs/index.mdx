## 先看免费额度

最后核对日期：2026-06-18。Cloudflare 的价格、额度和计划边界会变化，本站数字以官方 pricing / limits 为最终依据。

这个 Playbook 先回答一个问题：**一个普通项目能不能 0 元跑起来。**

### 先记这 12 个数字

用户最关心的不是产品名，而是“免费能用到什么程度”。先把这些数字记住，后面的选型会轻松很多。

| 关键数字 | 代表什么 | 普通项目怎么用 |
| --- | --- | --- |
| 静态资产请求免费且不限量 | Workers Static Assets / Pages 命中静态资源时不消耗 Worker 请求。 | 文档站、官网、博客、前端 bundle 先放静态层。 |
| 100,000 requests/day | Workers Free 每日动态请求额度，UTC 0 点重置。 | 小 API、Webhook、评论、表单先用它验证。 |
| 10 ms CPU/invocation | Workers Free 单次 CPU 额度。 | I/O 型接口通常够用；SSR、解析大文件、AI 前处理要谨慎。 |
| 5 USD/month | Workers Paid 账号级最低月费。 | 不是 Cloudflare Pro，也不是账单封顶；它买的是开发者平台动态能力。 |
| 10M requests/month | Workers Paid 已包含的动态请求量。 | 公开 API 或评论系统稳定有人用后，再考虑升级。 |
| 30M CPU ms/month | Workers Paid 已包含的 CPU 用量。 | SSR、队列消费者、定时任务和数据处理要一起看 CPU。 |
| 5M rows read/day | D1 Free 每日读取行数。 | 评论、反馈、小后台够起步；全表扫描会很快浪费额度。 |
| 100k rows written/day | D1 Free 每日写入行数。 | 写入型功能要做 Turnstile、限流和去重。 |
| 10 GB-month | R2 Standard Free storage。 | 附件、图片原图、导出文件先放 R2，不塞进站点包。 |
| 1M / 10M operations/month | R2 Free 每月 Class A / Class B 操作。 | 上传看 Class A，下载和读取看 Class B。 |
| 10,000 Neurons/day | Workers AI Free 每日免费分配。 | AI 搜索和摘要先做小模型、短输出、缓存和限流。 |
| 20,000 queries/month | AI Search 新实例 Free 查询额度，open beta 内免费。 | 文档搜索先 Pagefind；自然语言搜索确定有价值后再上 AI Search。 |

多数情况下可以。文档站、官网、博客、小 API、Webhook、评论、反馈、轻后台、文件下载和 AI 搜索原型，都可以先从 Free 额度里验证。关键不是把所有产品都打开，而是让流量停在最便宜的位置：

| 资源类型 | 先放哪里 | 为什么 |
| --- | --- | --- |
| 文档、官网、博客、前端资源 | Workers Static Assets / Pages | 静态资产请求免费且不限量。 |
| 小 API、Webhook、表单、评论接口 | Workers Free | 100,000 requests/day 足够做早期验证。 |
| 评论、表单、小后台数据 | D1 | 5M rows read/day、100k rows written/day、5 GB storage。 |
| 图片、附件、导出文件 | R2 Standard | 10 GB-month、1M Class A、10M Class B，egress 免费。 |
| 配置、低频缓存、开关 | KV | 100k reads/day、1k writes/day、1 GB storage。 |
| 邮件、通知、导入、后处理 | Queues | 10,000 operations/day。 |
| 房间、会话、限流器、实时状态 | Durable Objects | 100,000 requests/day、13,000 GB-s/day。 |
| 后台、预览环境、内网工具 | Access + Tunnel | Zero Trust Free 适合小团队 PoC。 |
| 文档搜索 | Pagefind -> AI Gateway / AI Search | 先本地索引，AI 有价值后再接托管能力。 |

## 免费额度大全

这张表是首页速查版。完整产品级表格、计费解释和官方来源见 [免费与付费边界](/platform/free-paid/)。

| 能力 | Free 额度 / 免费边界 | 5 USD/month Workers Paid 是否相关 | 最容易撞墙 |
| --- | --- | --- | --- |
| DNS | Free / Pro / Business 不按 DNS queries 收费；新 Free zone 200 records/zone，旧 Free zone 1,000。 | 无关。DNS 属于 zone / account 基础能力。 | 记录数、团队权限、企业级 DNS 需求。 |
| CDN / Cache | 所有计划可用；Free 有 10 条 Cache Rules；Free / Pro / Business 最大可缓存文件 512 MB。 | 无关。Workers Paid 不增加 Cache Rules。 | 复杂缓存规则、大文件、频繁 purge。 |
| SSL/TLS | Universal SSL、Origin CA、Always Use HTTPS、HSTS、TLS 1.3 可在 Free 使用。 | 无关。证书高级能力看 zone plan 或 add-on。 | 自定义证书、Keyless SSL、复杂证书治理。 |
| DDoS Protection | 所有计划都有 standard, unmetered DDoS protection。 | 无关。高级 DDoS 配置看 Enterprise / add-on。 | 源站 IP 暴露、应用层恶意写入、误伤排查。 |
| WAF Custom Rules | Free 5 条；Pro 20；Business 100。 | 无关。Workers Paid 不增加 WAF 规则。 | 规则数量、事件历史、Bot / API 高级防护。 |
| Rate Limiting Rules | Free 1 条；Pro 2；Business 5。 | 无关。它跟 WAF / zone plan 走。 | 登录、评论、上传、Webhook 入口太多。 |
| Turnstile | Free 20 widgets/account、10 hostnames/widget、unlimited challenges。 | 无关。Enterprise 才提升 widget / hostname / analytics 边界。 | 多域名、多租户、品牌和高级 Bot 检测。 |
| Zero Trust / Access / Tunnel | Free 适合 50 users 内的小团队；Tunnel 可发布 public hostname。 | 无关。Zero Trust seat 和日志另看计划。 | 用户数、Gateway、DLP、设备姿态、审计日志。 |
| Web Analytics | 免费；代理站点不限，非代理站点 10 个。 | 无关。 | 复杂归因、长期原始日志、跨系统分析。 |
| Workers Static Assets | 静态资产请求免费且不限量；存储 Assets 不额外收费。 | 相关但不收费。Paid 后静态请求仍免费且不限量。 | `run_worker_first`、SSR、API 路径误打进 Worker。 |
| Pages | Free 500 builds/month、1 concurrent build、100 custom domains/project、20,000 files/site、25 MiB/file。 | Pages Functions 按 Workers 计费。纯静态不需要先付。 | 构建次数、文件数、Functions 请求。 |
| Workers 请求 | Free 100,000 requests/day，UTC 0 点重置。 | Paid 含 10M requests/month，超出 0.30 USD/million。 | 公开 API、评论、搜索代理、SSR 动态请求。 |
| Workers CPU | Free 10 ms CPU/invocation。 | Paid 含 30M CPU ms/month；单次默认 30s，可设到 5min。 | SSR、AI 前处理、批量解析、大 JSON。 |
| Workers Logs | Free 200,000 events/day，保留 3 天。 | Paid 含 20M events/month，保留 7 天。 | 生产排障、长期审计、日志过量。 |
| Workers Builds | Free 3,000 build minutes/month、1 concurrent build。 | Paid 6,000 build minutes/month、6 concurrent builds。 | 多分支预览、频繁构建、大型站点。 |
| D1 | 5M rows read/day、100k rows written/day、5 GB total storage。 | Paid 含 25B rows read/month、50M rows written/month、5 GB。 | 无索引查询、写入增长、报表扫描。 |
| KV | 100k reads/day、1k writes/day、1k deletes/day、1k lists/day、1 GB storage。 | Paid 含更高月度读写删列和 1 GB。 | 写多、list 多、强一致需求。 |
| R2 Standard | 10 GB-month、1M Class A/month、10M Class B/month；egress 免费。 | 无需 Workers Paid 才能用；超出按 R2 自己计费。 | 热点读取、海量小文件、Infrequent Access 误用。 |
| Queues | 10,000 operations/day，消息保留 24 小时。 | Paid 含 1M operations/month，保留期可到 14 天。 | 重试、批处理、死信、消息体超过 64 KB。 |
| Durable Objects | SQLite-backed DO 可用；100,000 requests/day、13,000 GB-s/day。 | Paid 含 1M requests/month、400,000 GB-s/month。 | WebSocket 未 hibernation、对象常驻、连接规模。 |
| Durable Objects SQLite storage | 5M rows read/day、100k rows written/day、5 GB total。 | Paid 含 25B rows read/month、50M rows written/month、5 GB-month。 | 把 DO 当全站数据库或分析库。 |
| Hyperdrive | 100,000 database queries/day。 | Paid 为 unlimited database queries。 | 外部数据库慢查询、连接池、源库本身成本。 |
| Workflows | Free 100,000 requests/day、10 ms CPU/invocation、1 GB storage。 | Paid 含 10M requests/month、30M CPU ms/month、1 GB storage。 | 长流程、重试、状态增长。 |
| Analytics Engine | 100,000 data points/day、10,000 read queries/day。 | Paid 含 10M data points/month、1M read queries/month。 | 把指标系统当事务数据库。 |
| Workers AI | 10,000 Neurons/day 免费分配。 | 超过每日免费分配需要 Workers Paid，再按 Neurons 计费。 | 大模型、长输出、开放式无缓存调用。 |
| AI Gateway | 核心功能免费；Free persistent logs 为所有 gateway 合计 100,000 logs。 | Paid 为 10M logs/gateway，并可用 Logpush。 | 日志留存、模型成本、无上限调用。 |
| AI Search | 2026-04-16 后新实例 open beta 内免费；Free 20,000 queries/month、500 crawled pages/day。 | Paid 提升实例、文件、查询和爬取边界。 | 文档量、自然语言查询、AI Gateway / Workers AI 叠加用量。 |
| Vectorize | 30M queried vector dimensions/month、5M stored vector dimensions。 | Paid 含更高维度包。 | 文档未整理好就上向量库。 |
| Images | 5,000 unique transformations/month。 | Images Paid 另算，不由 Workers Paid 包掉。 | 原图存储、转换量、图片交付量。 |
| Browser Run | 10 minutes/day browser hours、3 concurrent browsers。 | Paid 含 10 hours/month。 | 截图、PDF、网页抓取不关闭 session。 |
| Realtime SFU / TURN | Cloudflare 到客户端出站前 1,000 GB/month 免费；STUN 免费且不限量。 | 超出按 realtime 产品计费，不是 Workers Paid 包掉。 | 音视频出站流量、并发会议、TURN 用量。 |
| Budget alerts | Pay-as-you-go 账号可按美元阈值邮件提醒 usage-based spend。 | 付费前必须开，但它不是封顶。 | 以为提醒会自动暂停服务。 |

## 0 元项目组合

免费额度要按项目组合读，不要按产品名背。

| 项目 | Free 起步组合 | 先盯哪个数字 |
| --- | --- | --- |
| 开源文档站 | Workers Static Assets / Pages + Pagefind + Web Analytics | 构建次数、文件数、静态资产是否误进 Worker。 |
| 个人官网 / 博客 | DNS + CDN + Universal SSL + Static Assets | 缓存命中、图片体积、构建次数。 |
| 小 API / Webhook | Workers Free + D1 + KV + Turnstile | requests/day、CPU、D1 写入、KV 写入。 |
| 评论 / 反馈系统 | Workers + D1 + Turnstile + WAF Rate Limiting | D1 写入、刷量、日志留存。 |
| 文件上传 / 下载 | R2 Standard + 签名上传 + CDN cache | storage、Class A、Class B。 |
| 小团队后台 | Access + Tunnel + Zero Trust Free | 50 users、Access policy、审计需求。 |
| AI 搜索试验 | Pagefind + AI Gateway + Workers AI / AI Search | Neurons、queries、Gateway logs。 |
| 实时小应用 | Durable Objects + WebSocket hibernation + Queues | DO requests、GB-s、Queues operations。 |

## 什么时候付 5 USD/month

Workers Paid 是账号级订阅，不是 Cloudflare Pro，也不是账单封顶。它最适合在 Free 已经证明项目有人用之后再开。

| 该付的信号 | 为什么 |
| --- | --- |
| Worker 请求稳定接近 100,000/day。 | Free 会在日额度耗尽后出问题；Paid 改成月度 included usage。 |
| CPU 经常超过 10 ms。 | SSR、AI 前处理、批处理和大 payload 解析需要更高 CPU 上限。 |
| 日志 3 天不够排障。 | Paid Workers Logs 留存和月度额度更适合生产。 |
| D1、KV、Queues、Durable Objects 已经是核心功能。 | 这些产品在 Paid 下进入更高月度额度和付费扩展。 |
| 需要更多 Cron、Subrequests、Worker 数量或更大 bundle。 | 这是开发者平台工程化能力，不是单次流量问题。 |

| 不该误会 | 正确理解 |
| --- | --- |
| 5 USD/month 不是 Cloudflare Pro。 | 它不增加 WAF、Cache Rules、Bot、证书或 zone plan 权益。 |
| Static Assets 不是所有请求免费。 | 命中静态资产才免费；SSR、Functions、API、`run_worker_first` 会走 Workers 计费。 |
| R2 egress 免费不是 R2 全免费。 | storage、Class A、Class B 仍然要看。 |
| Budget alerts 不是硬封顶。 | 它只发邮件，不会暂停服务或阻止继续计费。 |
| AI 免费额度不是公开无限用。 | 必须做缓存、限流、模型选择和日志观察。 |
