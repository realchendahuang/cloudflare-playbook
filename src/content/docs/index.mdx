## 先别看产品，先看免费额度

最后核对日期：2026-06-18。

Cloudflare 的价格、额度和计划边界会变化。本站数字以官方 pricing / limits 为最终依据；首页先把起步项目最该看的免费额度摊开，完整口径见 [免费额度大全](/platform/free-paid/)。

这个 Playbook 不从产品名开始，而是从免费额度开始。免费额度不是附录，它决定一个普通项目最初怎么设计。读者真正关心的是四件事：

- 项目能不能先 0 元跑起来。
- 0 元能跑到什么程度。
- 跑到哪里才该付费。
- 付的那 5 美元到底买到什么。

| 先问 | 直接答案 |
| --- | --- |
| 文档站、官网、博客能不能 0 元跑？ | 可以。静态资产请求免费且不限量，先用 Workers Static Assets / Pages。 |
| 小 API、评论、表单能不能 0 元验证？ | 可以。Workers Free、D1、KV、R2、Queues、Durable Objects 都有免费层。 |
| 免费阶段最该防什么？ | 防公开写入口被刷、防静态流量误进 Worker、防数据库全表扫描、防 AI 调用不设限。 |
| 什么时候付 5 USD/month？ | 动态请求、CPU、日志、队列、D1/KV/DO 等开发者平台能力进入真实生产后再付。 |

</Aside>

## All the fun：0 元先跑起来

免费额度最值得玩的地方，是它能组合成真实项目，不只是单个产品的试用期。先把下面这些组合跑通，再决定要不要付费。

| 项目 | 0 元起步组合 | 先盯哪个数字 |
| --- | --- | --- |
| 开源文档站 | Workers Static Assets / Pages + Pagefind + Web Analytics | 构建次数、文件数、静态资产是否误进 Worker。 |
| 教程站轻社区 | Workers Static Assets + Twikoo Cloudflare + D1 + Turnstile | D1 写入、刷量、日志留存。 |
| 个人官网 / 博客 | DNS + CDN + Universal SSL + Static Assets | 缓存命中、图片体积、构建次数。 |
| 小 API / Webhook | Workers Free + D1 + KV + Turnstile | 100,000 requests/day、10 ms CPU、写入口滥用。 |
| 表单 / 反馈箱 | Workers + D1 + Turnstile + Workers Logs | D1 rows written、Workers Logs 3 天留存。 |
| 文件上传 / 下载 | R2 Standard + 签名上传 + CDN cache | 10 GB-month、Class A、Class B。 |
| 小团队后台 | Access + Tunnel + Zero Trust Free | 50 users、后台入口、审计需求。 |
| AI 搜索原型 | Pagefind + AI Gateway + Workers AI / AI Search | Neurons、queries、Gateway logs。 |
| 实时小应用 | Durable Objects + WebSocket hibernation + Queues | DO requests、GB-s、Queues operations。 |

## Cloudflare 免费额度大全

这张速查表先回答“免费能不能跑”和“5 美元有没有帮助”。完整产品级表格、计费解释和官方来源见 [免费额度大全](/platform/free-paid/)。

| 能力 | Free 额度 / 免费边界 | 5 USD/month Workers Paid 是否相关 | 最容易撞墙 |
| --- | --- | --- | --- |
| DNS | Free / Pro / Business 不按 DNS queries 收费；新 Free zone 200 records/zone，旧 Free zone 1,000。 | 无关。DNS 属于 zone / account 基础能力。 | 记录数、团队权限、企业级 DNS 需求。 |
| CDN / Cache | 所有计划可用；Free 有 10 条 Cache Rules；Free / Pro / Business 最大可缓存文件 512 MB。 | 无关。Workers Paid 不增加 Cache Rules。 | 复杂缓存规则、大文件、频繁 purge。 |
| SSL/TLS | Universal SSL、Origin CA、Always Use HTTPS、HSTS、TLS 1.3 可在 Free 使用。 | 无关。证书高级能力看 zone plan 或 add-on。 | 自定义证书、Keyless SSL、复杂证书治理。 |
| DDoS Protection | 所有计划都有 standard, unmetered DDoS protection。 | 无关。高级 DDoS 配置看 Enterprise / add-on。 | 源站 IP 暴露、应用层恶意写入、误伤排查。 |
| WAF Custom Rules | Free 5 条；Pro 20；Business 100。 | 无关。Workers Paid 不增加 WAF 规则。 | 规则数量、事件历史、Bot / API 高级防护。 |
| Rate Limiting Rules | Free 1 条；Pro 2；Business 5。 | 无关。它跟 WAF / zone plan 走。 | 登录、评论、上传、Webhook 入口太多。 |
| Turnstile | Free 20 widgets/account、10 hostnames/widget、unlimited challenges。 | 无关。Enterprise 才提升 widget / hostname / analytics 边界。 | 多域名、多租户、品牌和高级 Bot 检测。 |
| Zero Trust / Access / Tunnel | Free 适合 50 users 内的小团队；Tunnel 可发布 public hostname。 | 无关。Zero Trust seat 和日志另看计划。 | 用户数、Gateway、DLP、设备姿态、审计日志。 |

## 什么时候付 5 USD/month

Workers Paid 是账号级订阅，不是 Cloudflare Pro，也不是账单封顶。它最适合在 Free 已经证明项目有人用之后再开。

| 该付的信号 | 为什么 |
| --- | --- |
| Worker 请求稳定接近 100,000/day。 | Free 会在日额度耗尽后出问题；Paid 改成月度 included usage。 |
| CPU 经常超过 10 ms。 | SSR、AI 前处理、批处理和大 payload 解析需要更高 CPU 上限。 |
| 日志 3 天不够排障。 | Paid Workers Logs 留存和月度额度更适合生产。 |
| D1、KV、Queues、Durable Objects 已经是核心功能。 | 这些产品在 Paid 下进入更高月度额度和付费扩展。 |
| 需要更多 Cron、Subrequests、Worker 数量或更大 bundle。 | 这是开发者平台工程化能力，不是单次流量问题。 |

| 不该误会 | 正确理解 |
| --- | --- |
| 5 USD/month 不是 Cloudflare Pro。 | 它不增加 WAF、Cache Rules、Bot、证书或 zone plan 权益。 |
| Static Assets 不是所有请求免费。 | 命中静态资产才免费；SSR、Functions、API、整站先跑 Worker 会走 Workers 计费。 |
| R2 egress 免费不是 R2 全免费。 | storage、Class A、Class B 仍然要看。 |
| Budget alerts 不是硬封顶。 | 它只发邮件，不会暂停服务或阻止继续计费。 |
| AI 免费额度不是公开无限用。 | 必须做缓存、限流、模型选择和日志观察。 |
