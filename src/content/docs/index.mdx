## 免费额度先看这里

最后核对日期：2026-06-18。Cloudflare 的价格、额度和计划边界会变化，本站所有数字都以官方 pricing / limits 为最终依据。

普通项目最先需要的不是产品百科，而是预算判断：哪些东西可以免费起步，哪些东西会先撞额度，什么时候该付 `$5/month` 的 Workers Paid，哪些能力必须看 zone plan、add-on 或 Enterprise。

| 第一眼先看 | 免费额度 / 免费边界 | 直接判断 |
| --- | --- | --- |
| 静态流量 | Workers Static Assets / Pages 静态资产请求免费且不限量。 | 文档站、官网、博客和前端页面先放静态层。 |
| 动态入口 | Workers Free 100,000 requests/day，10 ms CPU/invocation。 | 评论、Webhook、表单、小 API 可以先免费验证。 |
| 关系数据 | D1 Free 5M rows read/day、100k rows written/day、5 GB storage。 | 评论、表单、小后台、轻量 SaaS 先用 D1。 |
| 文件对象 | R2 Standard 10 GB-month、1M Class A/month、10M Class B/month，egress 免费。 | 图片、附件、导出文件不要塞进站点包。 |
| AI 试验 | Workers AI 10,000 Neurons/day；AI Search 新实例 open beta 内免费。 | 搜索先 Pagefind；自然语言搜索再接 AI Gateway / AI Search。 |
| 预算提醒 | Budget alerts 只发邮件提醒 usage-based spend，不会自动封顶。 | 付费前先开提醒，代码和架构仍要限流。 |

| 判断 | 结论 |
| --- | --- |
| 文档站、官网、博客 | 优先 Workers Static Assets / Pages，静态资产请求免费且不限量。 |
| 小 API、评论、表单、Webhook | Workers Free + D1 / KV / R2 / Queues / Turnstile 可以先跑真实流量。 |
| 文件和图片 | 原文件优先 R2，公开访问叠 CDN cache，避免把大文件塞进站点包。 |
| 搜索和 AI | 少量文档先 Pagefind；自然语言搜索再评估 AI Search、Workers AI、AI Gateway。 |
| 第一次付费 | 通常是 `$5/month` Workers Paid，用来放大动态请求、CPU、日志和开发者平台额度。 |
| 成本控制 | Budget alerts 只是提醒；真正的控制靠静态优先、缓存、限流、队列、配额和权限。 |

## 免费额度大全

这张表放首页，就是为了让你第一眼判断一个项目能不能免费跑起来。更完整的计费解释、官方来源和 GitHub 源文件见 [免费与付费边界](/platform/free-paid/)。

| 能力 | Free 重点额度 | 付费后先看什么 | 普通项目用法 |
| --- | --- | --- | --- |
| DNS | Free / Pro / Business 不对 DNS queries 收费；新 Free zone 200 records/zone，旧 Free zone 1,000。 | 记录数、SLA、团队治理和企业支持。 | 域名先接 Cloudflare，Web 记录代理，邮件和验证记录保持 DNS-only。 |
| CDN / Cache | CDN 可用于所有计划；Free 有 10 条 Cache Rules；Free / Pro / Business 最大可缓存文件 512 MB。 | 更多 Cache Rules、更短 Edge TTL、复杂 purge、Cache Reserve。 | 静态 hash 资源长缓存，HTML 谨慎缓存，大文件放 R2。 |
| SSL/TLS | Universal SSL、Origin CA、Always Use HTTPS、HSTS、TLS 1.3 可在 Free 使用。 | Advanced Certificate Manager、Custom Certificates、Keyless SSL。 | 普通站默认 Full (strict)，源站用 Origin CA 或公开 CA。 |
| DDoS | 所有计划都有 standard, unmetered DDoS protection。 | 高级 DDoS override、企业告警、网络层合同能力。 | Web 入口设为 Proxied，隐藏源站 IP。 |
| WAF / Rate Limiting | Free 有 5 条 Custom Rules、1 条 Rate Limiting Rule、Free Managed Ruleset。 | 更多规则、更长事件历史、Bot / API 防护。 | 登录、后台、API、评论、上传先放最小规则。 |
| Turnstile | 20 widgets/account、10 hostnames/widget、unlimited challenges。 | Enterprise widgets、hostname、analytics lookback、品牌和设备能力。 | 表单、注册、登录、评论先接 Turnstile。 |
| Zero Trust / Access / Tunnel | Zero Trust Free 为 50 users；Tunnel 可发布 public hostname。 | 用户数、日志、Gateway、DLP、设备姿态、审计。 | 后台、预览环境、内网工具先用 Access + Tunnel。 |
| Web Analytics | 免费；代理站点不限，非代理站点 10 个。 | Web Analytics rules、外部数据管道、复杂归因。 | 文档站和官网先开它。 |
| 静态资产请求 | Workers Static Assets / Pages 静态资产请求免费且不限量。 | 文件数、单文件大小、构建并发、动态 Functions。 | 文档、官网、博客主流量留在静态层。 |
| Pages | 500 builds/month、1 concurrent build、100 custom domains/project、20,000 files/site、25 MiB/file。 | 构建次数、并发、文件数、Functions 成本。 | 纯静态站和 PR 预览够用；复杂 Worker 能力看 Static Assets。 |
| Workers 请求 | 100,000 requests/day，UTC 0 点重置。 | Workers Paid 10M requests/month included，超出按量。 | 小 API、评论、Webhook、表单先在 Free 验证。 |
| Workers CPU | 10 ms CPU/invocation。 | Workers Paid 30M CPU ms/month included，单次默认 30s，可设到 5min。 | SSR、AI 前处理、批任务要看 CPU，不只看请求数。 |
| Workers Logs | 200,000 log events/day，保留 3 天。 | Paid 20M log events/month included，保留 7 天。 | 免费阶段够查近期问题；生产日志要采样和脱敏。 |
| D1 | 5M rows read/day、100k rows written/day、5 GB storage。 | Paid 25B rows read/month、50M rows written/month、5 GB included。 | 评论、表单、小后台、轻量 SaaS 优先用它。 |
| KV | 100k reads/day、1k writes/day、1k deletes/day、1k lists/day、1 GB storage。 | Paid 月度读写删除列表包和按量扩展。 | 配置、缓存、低频索引；不要当强一致数据库。 |
| R2 Standard | 10 GB-month、1M Class A/month、10M Class B/month；egress 免费。 | 存储、Class A、Class B；Infrequent Access 另算。 | 图片、附件、导出文件、下载对象优先放 R2。 |
| Queues | 10,000 operations/day，消息保留 24 小时。 | Paid 1M operations/month included，保留期可到 14 天。 | 邮件、通知、导入、后处理先异步化。 |
| Durable Objects | SQLite-backed DO 可用；100,000 requests/day、13,000 GB-s/day。 | Paid 1M requests/month、400,000 GB-s/month included。 | 房间、会话、限流器、WebSocket 状态才放进去。 |
| Durable Objects SQLite storage | 5M rows read/day、100k rows written/day、5 GB total。 | Paid 25B rows read/month、50M rows written/month、5 GB-month included。 | 每个实体的小状态可以放，报表和全局查询不要放。 |
| Workers AI | 10,000 Neurons/day 免费分配。 | 超过每日分配需要 Workers Paid，之后按 Neurons 计费。 | 小模型、短输出、原型验证先跑起来。 |
| AI Gateway | 核心能力免费；Workers Free 持久日志 100,000 logs total。 | Paid 10M logs/gateway、Logpush、长期审计。 | 外部模型、Workers AI、缓存、限流和调用日志统一入口。 |
| AI Search | Open beta 新实例免费；Free 100 instances/account、20,000 queries/month、500 crawled pages/day；Workers AI 和 AI Gateway 另算。 | Paid unlimited queries / crawling。 | 文档多起来再试自然语言搜索。 |
| Vectorize | 30M queried vector dimensions/month、5M stored vector dimensions。 | Paid 更高维度包和按量扩展。 | RAG 价值明确后再用，早期搜索先 Pagefind。 |
| Images | 5,000 unique transformations/month。 | Images Paid、storage、delivery。 | 原图放 R2，需要裁剪、格式转换、响应式图时再接。 |
| Browser Run | 10 minutes/day browser hours、3 concurrent browsers。 | Paid 10 hours/month included，更多并发和小时数按量。 | 截图、PDF、动态网页抓取先小规模验证。 |
| Zaraz | 1,000,000 free Zaraz Events/month。 | 超出后看 paid events。 | 第三方脚本多起来后再治理。 |
| Budget alerts | Pay-as-you-go 账号可按美元阈值发邮件提醒 usage-based spend。 | 它不是硬封顶。 | 付费前先开提醒，业务侧仍要限流和配额。 |

第一条成本铁律：**主流量停在静态资产、CDN、R2 和缓存层，只让真正需要计算、写入、鉴权、排队或 AI 推理的请求进入计费路径。**

## $5 Workers Paid 买什么

Workers Paid 是账号级订阅，不是 Cloudflare Pro，也不是预算封顶。它适合在免费额度已经证明项目有效之后再开。

| 它放大的东西 | Free | Workers Paid / Standard |
| --- | --- | --- |
| 动态请求 | 100,000 requests/day。 | 10M requests/month included，超出 $0.30/million。 |
| CPU | 10 ms CPU/invocation。 | 30M CPU ms/month included，超出 $0.02/million CPU ms。 |
| Static Assets | 静态资产请求免费且不限量。 | 静态资产请求仍免费且不限量。 |
| Worker 数量 | 100。 | 500。 |
| Cron Triggers | 5/account。 | 250/account。 |
| Subrequests | 50/request。 | 默认 10,000/request。 |
| Worker bundle | 3 MB。 | 10 MB。 |
| Static Asset 文件数 | 20,000 files/Worker version。 | 100,000 files/Worker version。 |
| Workers Logs | 200,000 events/day，保留 3 天。 | 20M events/month included，保留 7 天。 |
| Trace Events Logpush | 不可用。 | 10M requests/month included，超出按量。 |
| 开发者平台联动 | D1、KV、Queues、Durable Objects 等有 Free 边界。 | 月度 included usage、更高限制或付费入口。 |

## 免费阶段组合

| 你想做什么 | 免费阶段优先组合 | 先盯哪个额度 |
| --- | --- | --- |
| 文档站、官网、博客 | Workers Static Assets / Pages + Pagefind + Web Analytics | 构建次数、文件数、静态资产大小。 |
| 小 API、Webhook、评论 | Workers Free + D1 Free + Turnstile + WAF 最小规则 | requests/day、CPU、D1 rows written、日志。 |
| 文件上传和下载 | R2 Standard Free + presigned URL + CDN cache | storage、Class A、Class B。 |
| 后台、预览环境、内网工具 | Zero Trust Free + Access + Tunnel | 50 users、Access policy、日志。 |
| 搜索和 AI 试验 | Pagefind + Workers AI + AI Gateway + AI Search Free | Neurons、AI Gateway logs、AI Search queries。 |
| 异步任务和实时状态 | Queues Free + Durable Objects Free + WebSocket hibernation | operations/day、DO requests、GB-s、SQLite rows。 |
