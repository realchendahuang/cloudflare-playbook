## 先别买，先把免费层用对

最后核对日期：2026-06-18。Cloudflare 的价格、额度和计划边界会变化，本站所有数字都以官方 pricing / limits 为最终依据。

这个仓库先回答一个很现实的问题：一个普通项目到底能不能先用 Cloudflare 免费跑起来。答案是可以，但顺序要对：**阅读流量留在静态资产和 CDN，文件进 R2，关系数据进 D1，配置缓存进 KV，强一致状态进 Durable Objects，慢任务进 Queues，真正需要计算的请求才进 Worker。**

免费额度不是让你把所有产品都打开，而是帮你先验证项目。先把下面这几类跑顺，再决定要不要升级到 `$5/month` Workers Paid、Pro / Business zone plan 或专项 add-on。

## 免费额度第一屏判断

| 场景 | 免费能做到 | 最容易撞墙 | 下一步 |
| --- | --- | --- | --- |
| 文档站、官网、博客 | 静态资产请求免费且不限量，搜索可先用 Pagefind。 | 文件数、构建次数、动态 Functions。 | 先用 Workers Static Assets 或 Pages。 |
| 小 API、Webhook、表单 | Workers Free 100,000 requests/day，D1 / KV / Turnstile 可一起起步。 | 10 ms CPU、D1 写入、日志留存。 | 动态请求稳定后再升 Workers Paid。 |
| 评论、轻社区、反馈墙 | Workers + D1 + Turnstile + 最小 WAF 规则可以验证互动价值。 | 写入、审核、恶意提交、日志追踪。 | 先做限流和审核，再考虑付费日志与队列。 |
| 文件、图片、附件、导出物 | R2 Standard 有 10 GB-month、1M Class A、10M Class B，egress 免费。 | storage、Class A、Class B。 | 大文件不要进 Git、站点包或数据库。 |
| 后台、预览环境、内网工具 | Zero Trust Free 50 users，Tunnel 可发布 public hostname。 | 用户数、日志、设备姿态、审计。 | 50 人以内优先 Access + Tunnel。 |
| 搜索和 AI 试验 | Pagefind 不打后端额度；Workers AI 有 10,000 Neurons/day。 | Neurons、AI Gateway logs、AI Search queries。 | 先验证搜索体验，再接 AI Search / Vectorize。 |
| 异步任务和实时状态 | Queues Free 10,000 operations/day，Durable Objects Free 可做强一致状态。 | 消息保留、DO requests、GB-s。 | 慢任务进 Queues，房间状态进 Durable Objects。 |
| 成本观察 | Budget alerts 可提醒 usage-based spend。 | 它不是硬封顶。 | 产品侧限流、配额、缓存和权限一起做。 |

## 免费额度大全

这张表放在首页，就是为了让你第一眼判断项目能不能免费起步。更完整的计费解释、官方来源和 GitHub 源文件见 [免费与付费边界](/platform/free-paid/)。

| DNS | Free / Pro / Business 不对 DNS queries 收费；新 Free zone 200 records/zone，旧 Free zone 1,000。 | 记录数、SLA、团队治理和企业支持。 | 域名先接 Cloudflare，Web 记录代理，邮件和验证记录保持 DNS-only。 |
| CDN / Cache | CDN 可用于所有计划；Free 有 10 条 Cache Rules；Free / Pro / Business 最大可缓存文件 512 MB。 | 更多 Cache Rules、更短 Edge TTL、复杂 purge、Cache Reserve。 | 静态 hash 资源长缓存，HTML 谨慎缓存，大文件放 R2。 |
| SSL/TLS | Universal SSL、Origin CA、Always Use HTTPS、HSTS、TLS 1.3 可在 Free 使用。 | Advanced Certificate Manager、Custom Certificates、Keyless SSL。 | 普通站默认 Full (strict)，源站用 Origin CA 或公开 CA。 |
| DDoS | 所有计划都有 standard, unmetered DDoS protection。 | 高级 DDoS override、企业告警、网络层合同能力。 | Web 入口设为 Proxied，隐藏源站 IP。 |

## 免费额度的使用规则

| 规则 | 为什么重要 |
| --- | --- |
| 静态资产命中才免费且不限量。 | Workers Static Assets 如果被 `run_worker_first` 或 SSR 路径先打进 Worker，就会消耗 Worker 免费请求额度，超额后可能返回 429。 |
| 免费额度不是预算封顶。 | Budget alerts 只发邮件提醒，不会自动暂停服务或停止按量计费。 |
| Workers Paid 不是 Cloudflare Pro。 | `$5/month` 主要放大开发者平台额度，不会自动增加 WAF、Bot、证书、Rules 或企业网络能力。 |
| R2 免 egress 不是所有维度免费。 | R2 仍按 storage、Class A、Class B 计费，热点下载要看 Class B。 |
| D1 看扫描行数，不只看返回行数。 | 没有索引的查询会读更多 rows read，免费层最容易被慢查询消耗。 |
| AI 先从搜索体验开始验证。 | Pagefind 不打后端额度；自然语言搜索确认有价值后，再接 AI Gateway、Workers AI、AI Search 或 Vectorize。 |

## $5 Workers Paid 买什么

Workers Paid 是账号级订阅，不是 Cloudflare Pro，也不是预算封顶。它适合在免费额度已经证明项目有效之后再开，第一笔 `$5/month` 买的是动态能力、日志能力和开发者平台余量。

| 它放大的东西 | Free | Workers Paid / Standard |
| --- | --- | --- |
| 动态请求 | 100,000 requests/day。 | 10M requests/month included，超出 $0.30/million。 |
| CPU | 10 ms CPU/invocation。 | 30M CPU ms/month included，超出 $0.02/million CPU ms。 |
| Static Assets | 静态资产请求免费且不限量。 | 静态资产请求仍免费且不限量。 |
| Worker 数量 | 100。 | 500。 |
| Cron Triggers | 5/account。 | 250/account。 |
| Subrequests | 50/request。 | 默认 10,000/request。 |
| Worker bundle | 3 MB。 | 10 MB。 |
| Static Asset 文件数 | 20,000 files/Worker version。 | 100,000 files/Worker version。 |
| Workers Logs | 200,000 events/day，保留 3 天。 | 20M events/month included，保留 7 天。 |
| Trace Events Logpush | 不可用。 | 10M requests/month included，超出按量。 |
| 开发者平台联动 | D1、KV、Queues、Durable Objects 等有 Free 边界。 | 月度 included usage、更高限制或付费入口。 |

| 它买不到的东西 | 需要另看什么 |
| --- | --- |
| WAF / Bot / Rules / 证书的 zone 计划权益 | Cloudflare Free / Pro / Business / Enterprise。 |
| Load Balancing、Argo、Spectrum、Images、Stream 等专项产品 | 对应 add-on、媒体用量或网络产品价格。 |
| Enterprise 日志、专线、自有 IP、Magic Transit | 企业合同和网络工程预算。 |
| 成本硬封顶 | 产品侧限流、配额、队列、缓存和预算提醒一起做。 |

## 免费阶段组合

| 你想做什么 | 免费阶段优先组合 | 先盯哪个额度 |
| --- | --- | --- |
| 文档站、官网、博客 | Workers Static Assets / Pages + Pagefind + Web Analytics | 构建次数、文件数、静态资产大小。 |
| 小 API、Webhook、评论 | Workers Free + D1 Free + Turnstile + WAF 最小规则 | requests/day、CPU、D1 rows written、日志。 |
| 文件上传和下载 | R2 Standard Free + presigned URL + CDN cache | storage、Class A、Class B。 |
| 后台、预览环境、内网工具 | Zero Trust Free + Access + Tunnel | 50 users、Access policy、日志。 |
| 搜索和 AI 试验 | Pagefind + Workers AI + AI Gateway + AI Search Free | Neurons、AI Gateway logs、AI Search queries。 |
| 异步任务和实时状态 | Queues Free + Durable Objects Free + WebSocket hibernation | operations/day、DO requests、GB-s、SQLite rows。 |
