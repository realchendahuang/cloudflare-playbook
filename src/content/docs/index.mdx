## 这是什么

Cloudflare Playbook 不是官方文档的搬运，而是一份面向真实项目的实践索引。它会把 Cloudflare 的产品能力翻译成普通开发者能使用的判断框架：什么时候用、怎么组合、怎么部署、怎么验证、有什么风险。

	<Card title="产品地图" icon="document">

	</Card>
	<Card title="实战案例" icon="add-document">

## 推荐阅读顺序

1. 先读 [学习路线](/start/)，知道这份文档站怎么用。
2. 再读 [产品地图](/platform/)，建立 Cloudflare 产品之间的关系。
3. 然后读 [架构模式](/architecture/)，把产品组合成可落地方案。
4. 最后进入 [实战案例](/recipes/)，按一个具体场景跑通。
