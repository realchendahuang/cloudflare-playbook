# 最佳实践

最佳实践页沉淀跨产品的原则，避免每个案例都重复解释。

## 当前主题

| 主题 | 解决的问题 |
| --- | --- |
| [安全边界](/best-practices/security/) | 域名、凭据、访问控制、写操作和生产资源保护 |
| [成本控制](/best-practices/cost/) | 如何避免缓存、AI、存储和请求量失控 |

## 基本原则

- 每个外部入口都要有清晰的访问边界。
- 每个写操作都要知道失败后怎么重试或回滚。
- 每个缓存都要知道过期策略和失效方式。
- 每个成本项都要知道触发条件和观测方式。
