# 成本控制

成本控制的核心不是省到不用，而是知道什么行为会触发计费，并在代码里设置边界。

## 常见成本来源

- Workers 请求量和执行资源。
- Pages Functions 请求量。
- D1 读写和存储。
- R2 存储、请求和数据访问模式。
- AI 推理、向量检索和网关调用。
- 日志、分析和外部监控服务。

## 最小实践

- 给公开 API 加缓存和限流。
- 给 AI 接口加鉴权和用量限制。
- 给后台任务加队列和重试上限。
- 给大文件上传设置大小、类型和权限限制。
- 给每个案例写清“如何观察成本是否异常”。
