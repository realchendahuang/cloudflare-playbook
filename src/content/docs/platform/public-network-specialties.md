## 适用场景

| 问题场景 | 推荐能力 |
| --- | --- |
| 给设备或团队终端换 DNS resolver。 | 1.1.1.1。 |
| 做行业研究、趋势报告、安全背景分析。 | Radar。 |
| 服务器、设备、签名和日志依赖可靠时间。 | Time Services。 |
| 业务确实需要 IPFS 或 Ethereum RPC。 | Web3 Gateways。 |
| 面向中国大陆用户做合规和性能交付。 | China Network。 |
| 已经使用 Google 广告 / 分析标签。 | Google tag gateway。 |
