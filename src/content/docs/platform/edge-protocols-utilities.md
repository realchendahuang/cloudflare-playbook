## 使用信号

浏览器侧网络失败上报只能补充用户侧网络诊断，不替代 Workers Logs、Web Analytics 或业务日志。公开可验证随机数不能拿来生成密钥、访问凭证或会话密钥。代管客户 Cloudflare 账号面向合作伙伴场景，早期 SaaS 先看普通多租户。

企业配置版本管理适合企业域名配置灰度。隐私代理基础设施不替代 Access、Tunnel、WAF，它们仍然按各自场景使用。新媒体协议实验里，视频先看 Stream，实时音视频先看 Realtime。助手长期记忆没有明确场景时，先用 D1 / R2 / Vectorize。

## 可以先看

Cloudflare 资源变多，想按环境、团队、成本中心归类时，可以看资源标签。它可用于治理工具，但不需要在第一个 Worker 上引入。

用户反馈“打不开”，但服务端没有请求记录时，可以看网络失败上报。它只用于补充诊断用户侧网络问题。

做公开抽签或协议随机数，需要可验证随机源时，可以看公开随机源。它只用于公开可验证流程，不用于私密凭证。
