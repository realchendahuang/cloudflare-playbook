# R2 签名上传

这个案例用于演示如何让用户安全上传文件到 R2，并避免把存储桶直接暴露给公网。

## 适合场景

- 用户头像和附件。
- 导出文件。
- 私有下载资源。
- 静态大文件分发。

## 架构

```text
客户端
  │ 请求上传授权
  ▼
Worker
  │ 校验身份和文件策略
  ▼
R2
```
